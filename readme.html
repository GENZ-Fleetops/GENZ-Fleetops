Hosting

  • Netlify + GitHub: The repository root includes netlify.toml pointing at this folder (zip files/Zip 1/files). Push the whole repo to GitHub; Netlify deploys on each push. No build command. If you edit Netlify UI settings, set Publish directory to the same path.
  • Netlify manual: Drag the folder that contains all pages + assets/ into a manual deploy.
  • cPanel / shared hosting: Upload into public_html (or your web root). Set index.html as the default document.
  • Local: Open index.html in a browser; all module links use relative paths.
